## Abstract

<!-- TODO: Replace with paper abstract -->
Quantitative Inductive Machines (QIM) enable sequence-to-sequence regression
using language model architectures. By representing numerical data as token
sequences, QIM can be pretrained on large corpora of (input, output) pairs and
fine-tuned to new regression tasks with minimal data. This approach unifies
disparate prediction problems — from scientific simulation to performance
forecasting — under a single, flexible framework.
